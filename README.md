## Overview
Jet One is a sleek and modern website designed for luxury private aviation services. The website showcases private jet charters, corporate and group flights, helicopter charters, and VIP air travel services. With a clean design and responsive layout, it ensures a premium user experience on all devices.

The website includes:
- A dynamic carousel highlighting key services.
- Detailed sections for different aviation services.
- A private flight request form to submit travel details.
- A fully responsive navbar and footer with contact information.

---

## Features
- **Responsive Design:** Works on desktop, tablet, and mobile screens.
- **Carousel Slider:** Highlights luxury services with captions.
- **Service Sections:** Detailed descriptions and images for Private Jets, Corporate Flights, and Helicopter Charters.
- **Flight Request Form:** Users can submit travel details directly from the site.
- **Dropdown Menu:** Easy navigation for multiple service options.
- **Interactive Elements:** Hover effects on buttons, links, and dropdown items.
- **Footer:** Quick links, service links, and contact information for users.

---

## Technologies Used
- **HTML5** – Structure and content of the website.
- **CSS3** – Styling, layout, and responsive design.
- **Bootstrap 5** – Grid system, carousel, forms, and responsive components.
- **JavaScript (Bootstrap bundle)** – Carousel functionality and dropdown behavior.
